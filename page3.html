Kachiku 64

A 64 player local co-op party game

64 farm animals fighting to win through a series of mini games: tractors, planes, combine harvesters and ofcorse B.O.B!

Working as a large game development studio, around 30 people were tasked with realising the dream of a previous game studio. Taking the initial code base and assets and using them to improve upon and finish the game. 

My sub-group was tasked with creating a system to handle all bomb interactions within the game. This meant a hand in almost all game modes including the main gamemode. Inspired by bomberman and with a new character added to the game, the "BOB Drop" gamemode was created.
