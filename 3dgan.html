Generating 3D models using Deep Learning

A generative adversarial network which can generate unique 3D models

Using Tensorflow and Python, I created a GAN which can analyse and train from 3D voxel models.

Deep learning can be used for many different applications, from recognising numbers to generating entire games. This project uses Tensorflow and Tensorlayer to generate 3D models from a dataset. In this case, these models were cars.

Training this GAN took over 24 hours and around 3000 epocs to create the immages below. These are fully useable game assets (if a little un-defined)
