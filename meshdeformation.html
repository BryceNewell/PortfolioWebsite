Simulation Metalwork in Virtual Reality

Deforming meshes in real-time within VR.

Using the Oculus Rift and Unity, a simulation was created to deform meshes at real time, reacting to the players inputs, given by the full motion Touch controllers, giving a more realistic feel to crafting in VR games.

Crafting systems are a great tool in many games but the idea of simulation style crafting in Virtual Reality has never been fully explored in games. For this reason, I set about to create a tool for use in a wider game which allows the user to hit ingots of metal with a hammer in order to create unique items.
