Open World Level Streaming

Chunk-like streaming of an open world in unity.

Using data streaming techniques to load an open world in an area around the player, focusing on optimisation of this system

Using Unity and C#, I created a system to load and unload scenes around the player as they run around. Large items are stored as Unity scenes while smaller items are stored within XML files.

There are also NPC's within the simulation. These will wander around between points in any of the active scenes. They search for the player and will attack if they are close enough. If no player is around then they will move to a location and perform an action there. The NPC's also save to an XML file and unload and load within the scene they currently reside.
